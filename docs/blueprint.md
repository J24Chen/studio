# **App Name**: Rabbit & Steel Companion

## Core Features:

- Item Database: Comprehensive database of all Rabbit & Steel items.
- Class-Specific Tier Lists: Tier lists of items, customized for each playable class, generated using user ratings and AI analysis.
- Item Search: Quickly search for items by name or keyword.
- Detailed Item Information: Display in-depth item descriptions and stats, pulled from the game's files. Show how the item stats synergize with a specific class's kit when used as a tool to generate tier estimates.
- User Rating System: Allow users to rate items based on their effectiveness with specific classes.
- Tier List Creation and Viewing: Let users create, view, and share tier lists for different classes, sorted using AI
- Database Storage: Persistently store the items

## Style Guidelines:

- Primary color: Dark, blackish/gray with purple accents, reflecting the magical nature of the game.
- Background color: blackish-gray
- Accent color: a dark purple to draw the eye to important elements, with accents of black.
- Headline font: 'Playfair', a modern serif with an elegant, fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif that pairs well with Playfair for longer text.
- Custom icons representing item categories and classes.
- Subtle animations for item transitions and tier list updates.
